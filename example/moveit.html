Moveit.js

Tiny and awesome jQuery plugin that provides smooth draggable ability.

Example

How to Use?

1 - Include Script

<script src="path/to/moveit.min.js"></script>

2 - Dispatch

							$(".moveable").moveable();

3 - Set Listeners

							$(".moveable").on("moveit:start", function(event) {
							  // Do Your Needs..
							});

Available Events:

moveit:start moveit:move moveit:stop
